Le HTML, c'est quoi ?

  • Hyper Text Markup Language
  • Langage de balises
  • Syntaxe basée sur le XML
  • Réglementé par le W3C

La balise

  • Mot clé: div, body, footer
  • Mot clé entre chevrons
  • Peut être utilisé par paire ou orpheline
  • paire: <div>HTML for newbies</div>
  • orpheline: <br/>

La balise

On peut rajouter de l'information aux balises grâce aux attributs.

Les attributs peuvent permettre:

  • D'identifier une balise: id
  • D'identifier un groupe de balises: class
  • Donner des informations spécifiques: href, src
  • <img id="logo" src="logo.png" alt="Logo du site">
  • <a href="https://github.com/Mcdostone/HTML-CSS" target="_blank">Mon super dépot github</a>

Structure de base

<!doctype html>
<html lang="fr">

  <head>
    <!-- Configuration -->
    <meta charset="utf-8">
    <title>Windows sucks</title>
    <meta name="description" content="Why Windows sucks">
    <meta name="author" content="Bill Gates">
    <link rel="stylesheet" href="css/styles.css">  
  </head>
  
  <body>
    <!-- Contenu -->
  </body>
  
</html>

On peut découper le site en plusieurs parties:

  • Un menu en haut
  • Le menu contient plusieurs liens vers d'autres pages

  • Un bloc central contenant les gags
  • Un gag est composé d'un titre et d'une image

  • Un petit bloc à droite contenant d'autres gags
  • Ces autres gags sont composés d'une image et d'un titre

À vous de coder !

Lancez Sublime Text 2 (ou votre éditeur favori)

Ouvrez le fichier exercice-1.html dans le dossier exercices/

Ouvrez exercice-1.html avec votre navigateur web afin de visualiser la construction du document HTML !

Création du menu

  • Créer les liens cliquables "Trending" et "Fresh" en vous aidant du lien déjà crée
  • Imbriquer tous les liens crées dans une balise header 
    <header>...</header>

Création d'un gag

  • Créer un balise h3 qui contiendra le titre d'un gag.
  • Créer une balise img qui contient l'image associé au titre
  • Imbriquer le gag (titre + image) dans une balise div qui aura comme class la valeur "gag"
  • Faites cela pour toutes les titres et image qui suivent:
<!-- Titles & URLs -->

Création du principal container

  • Mettez toutes les div (gag) dans une nouvelle div avec l'id "container"

Création du nav

  • Créer un nouvelle balise nav à la suite de div#container
  • À l'intérieur de nav, vous devez listez d'autres gags.
  • Pour cela, utiliser le tag ul pour définir une liste et la tag li pour définir un item de cette liste.
  • Chaque li aura la class "tiny-gag". Cette li contiendra un titre h4 ainsi qu'un balise img représentant l'image.
