Le modèle de box ?

Le modèle de box

  • Chaque élément est en fait une boite rectangulaire
  • On distingue 2 types de boites:

  • inline: Les éléments peuvent être sur la même ligne (a)
  • La taille des boites inline dépend du contenu

  • block: Un seul block par ligne (h4, h3)
  • La hauteur dépend du contenu
  • Un block prend toute la largeur possible !

  • Exemple

Les règles CSS

  • X {
  display: block | inline | inline-block;
  width: 35%; /** 35% de l'élement parent */
  height: 40px;
}

Les marges

Il est possible d'appliquer des marges à l'intérieur (padding)

et à l'extérieur(margin) de notre box.

Ces marges peuvent s'exprimer en % ou en px.


/** Elément centré */
X {
  display: inline-block;
  width: 50%;
  margin-left: 25%;
  /** Marge intérieur de 10 px tout autour de mon élement */
  padding: 10px;
}

Les bordures

  • Une bordure est définie par une taille, une couleur et un type de bordure.
  • /** Elément centré */
X {
  display: inline-block;
  width: 50%;
  margin-left: 25%;
  border: 3px solid orange;
  padding: 10px;
}
  • Seulement il y a un problème !

Attention à la taille de chaque box !

Imaginons que l'on souhaite un élément qui prenne toute la largeur: 

X {
  display: inline-block;
  width: 100%;
  height: auto;
  border: 5px solid orange;
  padding: 10px;
}

Notre box va dépasser de (10*2 + 5*2) = 30px

La solution: box-sizing: border-box;

Exemples

/** Enleve les marges de base */
html, body {
  margin: 0;
  padding: 0;
}
header {
  display: block;
  width: 80%;
  /** S'adapte au contenu => 78px*/
  height: auto;
  margin: auto; /** Centre automatiquement */
}
header a {
  display: inline-block;
  padding: 4px 16px;
  /** 70px + 2*4 = 78px de hauteur */
  height: 70px;
  line-height: 70px;
}

Les floats

  • Autre manière de positionner les éléments
  • Un élément peut flotter à gauche ou à droite.
  • Un élément peut forcer un élément à ne pas flotter à coté de lui !
  • X {
  /** Flotte à gauche */
  float: left;
  /** Rien ne pas flotter à ma droite */
  clear: right;
}

Revenons au problème !
