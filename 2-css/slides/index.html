Le CSS, c'est quoi ?

  • Cascading Style Sheets
  • Langage informatique
  • Réglementé par le W3C
  • Appliquer des règles de mises en forme sur des éléments
  • => Mettre en forme les balises HTML

une règle CSS est composée:

  • d'un sélecteur
  • d'un ensemble de propriétés/valeurs

  • Le sélecteur permet de sélectionner le(s) élément(s) dans le document HTML via leur nom de balise, d'id ou de class
  • Les propriétés définissent les règles de mise en forme (couleur, taille, espacement ...)
  • /* Selector */
body { 
  /* properties */
  background-color: rgb(34, 49, 63);
  color: white;
}

Fonctionnement

  • une règle s'appliquant à un élément de l'arbre s'applique également à ses descendants.
  • lorsque plusieurs règles héritées peuvent s'appliquer, c'est la plus proche qui s'applique.

D'autres sélecteurs existent

  • /** Fond blanc pour le body et footer */

  background-color: white;

/** s'applique au p à l'intérieur de #container */
#container p {
  color: gray;
  font-size: 2em;
}

Quelques propriétés

* {
  color: red; /** Couleur du texte */
  background-color: #FFFFFF; /** Couleur d'arrère plan de l'élement */
  font-size: 18px; /** taille du texte */
  font-family: impact; /** Police à utiliser */
  text-align: center; /** Alignement du texte */
  opacity: 0.5; /** Opacité de l'élélement entre 0.0 et 1.0 */
}

À vous de coder !

  • Lancer votre éditeur de texte !
  • Ouvrer les fichiers dans le dossier exercices/

  • Dans un premier temps, on va créer notre fichier css. Créer un fichier style.css et enregistrer le dans le dossier exercices/
  • Il reste ensuite à importer le fichier CSS dans le fichier HTML en écrivant cette balise dans le fichier HTML: 
    <head>
  ...
  <link rel="stylesheet" type="text/css" href="style.css">
</head>
  • Insérer un petite règle dans ce fichier comme par exemple (afin de s'assurer que le fichier CSS est bien chargé dans le document HTML) 
    body {
  background-color: orange;
}

Maintenant, votre page HTML doit être de couleur orange !
